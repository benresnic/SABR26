## Optimizing Hitter Value Across Game States

This GitHub repository holds all code used for Team Resnic 2026 Diamond Dollars Case Competition at the SABR Analytics Conference. The prompt is about optimizing hitter value across game states: determening when, if, and why a subset of 27 hitters deemed "High-K / High-HR / Low Contact" should switch their season long "Hitter-Optimized Approach" to a situation based "Win-Creation Approach".
